# Culture Link

---

## サービス概要（※最後に書いてOK）

このアプリ は、  
**歴史・文化・人物・出来事をタイムラインで可視化しつながりや因果関係を理解する文化史学習アプリ**です。

受験生や定期テストを控えている学生で文化史を体系的に理解するのが難しい人が、  
**出来事同士のつながりを可視化しながら学べる環境**を提供します。

---

## このサービスの開発背景

私自身学生時代から歴史や文化に興味があり、大学では歴史学を専攻していました。  
本や記事、学校の授業などで様々な出来事を知る機会がありました。

しかし、

- 歴史の出来事
- 文化の発展
- 人物の影響

などが個別の知識として断片的に存在してしまい、  
「この出来事とこの文化はどう繋がっているのか？」  
という関係性を整理することが難しいと感じていました。

一方で、この文化史というのは受験でも必ず出題される分野で、得点するために時間を掛けて暗記で突破しがちになる分野になっています。 また、受験生や定期テストを控えている学生の時間は有限で、なるべく配点のウェイトや学習量の多い、国数英に時間を割きたいというのが本音です。
しかしながら、**文化に関する歴史の知識を体系的に蓄積する場所がない**ことで、自分自身で断片的に散りばめられた知識を体系化するために日本史や世界史に割く時間が必要になってしまい、学生の負担になってしまっています。私自身もそこは痛感していました。そこで、**出来事同士の関係性を整理して理解できる環境**が必要だと感じ、このアプリの開発に至りました。

## 想定しているユーザー

- ターゲット：受験生、定期テストを控えている学生
- 理由：受験や定期テストによって、文化史に触れなくてはいけないため。また受験生は必ず、年に一度存在しターゲットが必ず補充されるため。

## サービスの利用イメージ

1. ユーザーはなぜその文化が起こったのか、タイムラインを可視化することで因果関係を理解し学習を助けます。
2. ユーザー登録をすることで、学習予定を作成することができます。
3. 各単元ごとに小テストがあるので、小テストで理解の確認ができます。
4. お気に入りの単元を登録できます。
5. タイムラインにはキーワードがあり、キーワードをクリックするとそのキーワードの詳細を知ることができ、文化史だけでなく政治史なども補填することができます。

## サービスの差別化ポイント・推しポイント

他にも学習系のアプリは存在しますがこのアプリは以下の点で違います。

### 1. 「歴史全般」ではなく「文化史」という暗記になりがちな分野をメインにしています。

既存アプリは歴史全般を取り扱っていますが、このアプリは**文化史**という暗記になりがちで、教科書や資料集でも体系化されていない部分をメインにして、学生の学習の補助をします。

### 2. 「覚える」ではなく「つながり」の理解を重要視しています。

文化史はどうしてもその作品の作品名や作者を暗記だけして、テストや試験に臨みがちですが、このアプリはつながりを理解することで、どうしてその文化が生まれたのか、なぜその作品が制作されたのかつながを理解することで、暗記から脱却します。

### 3.タイムラインとつながりの可視化

他のアプリは一問一答で暗記になりがちで、自ら因果関係などを整理する必要がありますが、既にアプリ内でタイムラインとつながりが整理されているので、自らまとめる手間がなくなりユーザーの負担を減らします。

## ユーザーの獲得について

- **実際に学校の先生に使用してもらう**  
  　友達に先生がいるので、アプリを使用してもらえないか打診する。
- **X(twitter)で宣伝する**  
  　受験界隈向けにアプリの宣伝をする。

  ## データの管理について

- **基本的にwekipediaを使用する**  
  　自分の言葉で要約し、出典を記載すれば商用利用も可能なため。
  　掲載されている情報も一般論や他の論、参考記事の記載があり信憑性が高いため
- **画像についてはパブリックドメイン画像を使用**  
  　ウィキメディア・コモンズや美術館のデジタルコレクションを利用する。
