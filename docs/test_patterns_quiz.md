# 小テスト機能 手動テストパターン

## テスト準備
- テストアカウント: `test@example.com` / パスワードは設定済みのもの
- `./bin/dev` でサーバー起動済み
- seedデータが投入済み（ルネサンス / 古代ギリシャ / バロック / ロマン主義の4カテゴリ・各1〜3問）

---

## 1. 未ログイン状態

### 1-1. ヘッダー確認
- [ ] ヘッダーに「小テスト」リンクが表示されて**いない**こと

### 1-2. 直接URLアクセス
- [ ] `/quizzes` にアクセス → ログインページにリダイレクトされること
- [ ] `/quizzes/1` にアクセス → ログインページにリダイレクトされること
- [ ] `/quiz_results/1` にアクセス → ログインページにリダイレクトされること

---

## 2. ログイン〜ヘッダー

### 2-1. ログイン後のヘッダー確認
- [ ] ログイン後、ヘッダーに「小テスト」リンクが表示されること
- [ ] 「小テスト」をクリック → 小テスト一覧画面（`/quizzes`）に遷移すること

---

## 3. 小テスト一覧画面（`/quizzes`）

### 3-1. ページタイトル
- [ ] 「小テスト一覧 (Quiz List)」が表示されること
- [ ] 説明文「歴史と文化の知識を深めましょう」が表示されること

### 3-2. カテゴリフィルター
- [ ] カテゴリ選択ドロップダウンが表示されていること
- [ ] ドロップダウンに全カテゴリ（ルネサンス / 古代ギリシャ文化 / バロック / ロマン主義）が表示されていること
- [ ] カテゴリを選ぶ → そのカテゴリのクイズだけが表示されること
- [ ] カテゴリ未選択に戻す → 全クイズが表示されること

### 3-3. クイズカード表示
- [ ] 各クイズカードに以下が表示されること:
  - [ ] サムネ画像（未設定のものはグレーの背景）
  - [ ] カテゴリタグ（オレンジ背景）
  - [ ] クイズタイトル
  - [ ] 問題数「X問」
  - [ ] アクションボタン
- [ ] ルネサンス / ロマン主義はサムネ画像が表示されること
- [ ] その他は画像なし（グレー背景のみ）で表示されること

### 3-4. ステータス別ボタン表示
- [ ] 未受験のクイズ: 「テストを開始する」（オレンジ）が表示されること
- [ ] 完了済みのクイズ: 「もう一度受ける」（グレー）と右上に「完了」バッジが表示されること

### 3-5. ボタンからの遷移
- [ ] 「テストを開始する」をクリック → 実施画面（`/quizzes/:id`）に遷移すること
- [ ] 「もう一度受ける」をクリック → 同じく実施画面に遷移すること

### 3-6. ページネーション
- [ ] 6件を超える場合、ページネーションリンクが表示されること
- [ ] ページ番号をクリック → 該当ページに遷移すること
- [ ] カテゴリ絞り込み中も正しくページネーションされること

---

## 4. 小テスト実施画面（`/quizzes/:id`）

### 4-1. 画面表示確認
- [ ] クイズタイトルが表示されること
- [ ] 「第 1 / N 問」が表示されること（Nはそのクイズの総問題数）
- [ ] プログレスバーが `1/N` の位置まで塗られていること
- [ ] 最初の問題が表示されていること
- [ ] 2問目以降は非表示になっていること

### 4-2. 問題カード（画像あり：ルネサンスクイズ）
- [ ] 各問題ごとに異なる画像が表示されること
  - [ ] Q1「ルネサンス発祥の地」→ ルネサンスの絵画
  - [ ] Q2「モナ・リザを描いた画家」→ モナ・リザの画像
  - [ ] Q3「システィーナ礼拝堂」→ システィーナ礼拝堂の画像
- [ ] 画像右下に「出典: ...」の小さな表記があること
- [ ] 問題文が画像の下に表示されること

### 4-3. 問題カード（画像なし：古代ギリシャ / バロック / ロマン主義）
- [ ] 画像領域がなく、問題文のみがカード内に表示されること

### 4-4. 選択肢表示
- [ ] A/B/C/Dのラベル付きで選択肢が縦に並ぶこと
- [ ] 各選択肢は枠線付きカードで表示されること
- [ ] ホバーで枠線の色が変わること

### 4-5. 選択肢の選択
- [ ] 選択肢をクリック → 枠線が青く、背景が薄い青に変わること
- [ ] 別の選択肢をクリック → 選択が切り替わること（ラジオボタン動作）
- [ ] 同じ問題で複数の選択肢が同時選択状態にならないこと

### 4-6. ナビゲーションボタン
- [ ] 初期表示で「前の問題」ボタンが無効化（グレーアウト）されていること
- [ ] 「次の問題へ」ボタンは表示されていること
- [ ] 「採点する」ボタンは表示されていないこと

### 4-7. 次の問題への移動
- [ ] 「次の問題へ」をクリック → 次の問題が表示され、前の問題が非表示になること
- [ ] カウンタが「第 2 / N 問」に更新されること
- [ ] プログレスバーが進行すること
- [ ] 「前の問題」ボタンが有効化されること

### 4-8. 前の問題への移動
- [ ] 「前の問題」をクリック → 前の問題に戻ること
- [ ] カウンタ・プログレスバーが戻ること
- [ ] 前の問題で選択した選択肢がハイライトされたまま維持されていること

### 4-9. 最後の問題での表示切替
- [ ] 最後の問題に到達すると「次の問題へ」ボタンが非表示になり、「採点する」ボタンが表示されること
- [ ] 全問未回答の状態では「採点する」ボタンが無効化（グレーアウト）されていること

### 4-10. 全問回答後の採点ボタン有効化
- [ ] すべての問題に回答 → 「採点する」ボタンが有効化されること（オレンジ色）
- [ ] 1問でも未回答に戻すと無効化されること

---

## 5. 採点・結果保存（正常系）

### 5-1. 全問正解で採点
- [ ] ルネサンスクイズで全問正解を選択して「採点する」をクリック
- [ ] 結果画面にリダイレクトされること
- [ ] スコアが100%で表示されること
- [ ] 「素晴らしい！ (Excellent!)」のメッセージが表示されること
- [ ] 円形プログレスが緑色で塗られていること

### 5-2. 半分正解で採点
- [ ] ルネサンスクイズで半分ほど正解を選択して「採点する」
- [ ] スコアが正しく表示されること（3問中1問正解なら 33% など）
- [ ] スコアに応じたメッセージが表示されること

### 5-3. 全問不正解で採点
- [ ] 意図的に全問不正解を選択して「採点する」
- [ ] スコアが0%で表示されること
- [ ] 「復習しましょう (Let's review!)」のメッセージが表示されること

---

## 7. 結果画面（`/quiz_results/:id`）

### 7-1. サマリーカード
- [ ] 「テスト結果 (Quiz Results)」の見出しが表示されること
- [ ] クイズタイトルが表示されること
- [ ] 円形プログレスが表示され、中央にスコア（%）と「X / Y 正解」が表示されること
- [ ] スコア帯に応じたメッセージ（タイトル + 説明）が表示されること

### 7-2. 解説と振り返り
- [ ] 「解説と振り返り (Review)」の見出しが表示されること
- [ ] 各問題ごとにカードが表示されること
  - [ ] 正解の問題は左端に緑色のボーダー
  - [ ] 不正解の問題は左端に赤色のボーダー
- [ ] 各カードに以下が表示されること:
  - [ ] 「第 X 問」の連番
  - [ ] 「正解 / 不正解」バッジ
  - [ ] 問題文
  - [ ] 「あなたの回答」と「正解」の比較
  - [ ] 不正解の場合は「あなたの回答」に赤文字・取り消し線
  - [ ] 解説文（explanationが登録されている場合）
  - [ ] 右上に正誤アイコン（チェック / バツ）

### 7-3. アクションボタン
- [ ] 「もう一度挑戦する」（オレンジ）が表示されること
- [ ] 「小テスト一覧に戻る」（グレー）が表示されること

### 7-4. 「もう一度挑戦する」遷移
- [ ] 「もう一度挑戦する」をクリック → 同じクイズの実施画面に戻ること
- [ ] 選択状態はリセットされていること

### 7-5. 「小テスト一覧に戻る」遷移
- [ ] 「小テスト一覧に戻る」をクリック → 一覧画面（`/quizzes`）に戻ること
- [ ] 完了したクイズに「完了」バッジ・「もう一度受ける」ボタンが表示されていること
